# JhaTest.github.io
## Web 게시판 개인 프로젝트

1.회원제 사이트 구현
- 회원 가입
- 회원 로그인 & SNS 로그인
- 회원 탈퇴

2.게시판 구현
- 공지사항 게시판
- 자유게시판
- 검색 기능 추가제목에서 검색
- 내용에서 검색
- 작성자에서 검색
- 자기가 작성한 글만 수정 가능하게 처리
- 최신글 표시 기능 (최신글에 대한 기준은 본인이 정할 것: 하루, 삼일, 일주일 등...)
- 댓글 기능 

3.모든 코드에는 BootStrap 적용

4.관리자 기능
- 전체회원 조회
- 개별 회원 조회
- 회원 로그인 정지
- 회원 강제 탈퇴
- 게시글 삭제
